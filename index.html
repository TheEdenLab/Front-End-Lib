Front-End Lib

The ultimate front-end library for any website, eradicating the usual need for CSS and JavaScript!

How to Use:

Simply add this code inside the <head> tag of your document.

<link rel="stylesheet" type="text/css" href="https://amethysium.github.io/front-end-lib/1.0.0/style.css"> <script type="text/javascript" src="https://amethysium.github.io/front-end-lib/1.0.0/script.js"></script>
